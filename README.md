# MouseCatcher

MouseCatcher is a Lazarus (FreePascal/LCL) mouse diagnostic and testing tool.

It visualizes:

- Cursor movement with tail tracking
- Distance travelled
- Button status (Left, Middle, Right, Back, Forward)
- Wheel activity
- Click counters
- Capture state

## Purpose

Designed for diagnosing hardware issues with mice, RF dongles, scroll wheels and extra buttons.

## Requirements

- Lazarus 4.x
- FreePascal
- Linux Mint / Linux (tested)
- LCL widgetset
